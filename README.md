## 📦 Установка бека
```bash
# 1. Клонируйте репозиторий
$ git clone https://github.com/Russian-Gamedev-Arrising/cs-smokes.git

# 2. Перейдите в папку проекта
$ cd cs-smokes 
$ cd backend

# 3. Установите зависимости
$ pip install -r requirements.txt
```
